tagger.parser.base_event module¶

Base AWS Event Parser module

This module abstracts common attributes from all AWS Events to reduce duplicate code. This module is not called directly and is used mainly as an inheritance for more specific event parsers.

tagger.parser.ec2_event module¶

EC2 Event Parser module

This module parses the AWS Event dictionary for EC2 Events to return needed data.

Note

The module expects specific AWS Event Data, passed through the event parameter. It supports four events, each referring to a supported resource:

  • RunInstances
  • CreateSecurityGroup
  • CreateVolume
  • CreateImage
